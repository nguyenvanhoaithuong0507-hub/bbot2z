## ✨ Tầm nhìn

BBot được thiết kế như một **trung tâm điều khiển AI-first**: một giao diện duy nhất để theo dõi node, điều phối agent, đọc tài liệu, quản trị repo và chuẩn bị pull request tự động.

> BBot không chỉ là dashboard — đây là lớp điều phối giúp workflow DevSecOps rõ ràng, nhanh, an toàn và đẹp trên cả desktop lẫn mobile.

## 🧭 Tính năng nổi bật

| Mảng | Giá trị |
|---|---|
| **AI Console** | Trợ lý AI, prompt workflow và command palette cho thao tác nhanh. |
| **Telemetry** | Giám sát CPU/RAM, node active, uptime, latency và cảnh báo quá tải. |
| **Repository Hub** | Trình bày metadata repo, trạng thái, ngôn ngữ, stars/forks và tác vụ liên quan. |
| **PR Blueprint** | Không gian chuẩn bị pull request, kế hoạch nhánh và auto-run workflow. |
| **Mega Menu 20 trang** | Menu 3 gạch nhiều tầng nằm trên cùng bên trái, gom Core, AI, DevOps và Docs. |
| **Brand BBot** | Logo tím/xanh, hệ màu đồng bộ cho nút, badge, icon và trạng thái giao diện. |

## 🏗️ Kiến trúc nhanh

```text
bbot/
├─ public/bbot-logo.svg          # Logo chính dùng cho GitHub README và web app
├─ src/App.tsx                   # Giao diện BBot Console + mega menu 20 trang
├─ src/index.css                 # Theme, scrollbar và animation màu thương hiệu
├─ src/data.ts                   # Dữ liệu service/repository/docs
├─ server.ts                     # Express/Vite server
└─ README.md                     # Trang giới thiệu GitHub chuyên nghiệp
```

## ⚡ Chạy local

```bash
npm install
npm run dev
```

Sau đó mở địa chỉ local mà terminal hiển thị.
